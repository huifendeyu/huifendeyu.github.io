Git 冲突的解决

1、A写了A.js代码

2、A提交了

3、B也写了A.js

4、B也提交了

5、此时就会出现下图的冲突问题

解决方法

当B修改完冲突

1、B：git add .

2、B：git commit -m ‘修改冲突’

3、B：git push origin dev

4、A：在A分支 git pull origin dev

5、B：在B分支 git pull origin dev
